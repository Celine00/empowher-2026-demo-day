# EmpowHER 2026 Demo Day Deck

Static reveal.js deck for the EmpowHER 2026 Demo Day talk, "AI时代的感受力与良质".

## Preview

Open `index.html` in a browser. It redirects to `deck/`, which is the canonical presentation.

## GitHub Pages

This repo is ready to publish from the `main` branch root. After creating the GitHub repository, enable GitHub Pages with:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

The published URL will be:

```text
https://<owner>.github.io/<repo>/
```
